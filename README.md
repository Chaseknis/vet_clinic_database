# 📖 Vet Clinic Database <a name="about-project"></a>

- ***Vet Clinic Database is a comprehensive and efficient system that manages all the critical information for a veterinary clinic. Using PostgreSQL, I have created a database that allows for easy access and retrieval of important data.***

***This a starting point for someone that wants to learn how to create Databases and PostgreSQL Queries***

To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need:

- VSCode
- PostgreSQL
- NODE.Js
